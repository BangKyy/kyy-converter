Converter

The notion of conversion is a change from one knowledge system to another. In addition, conversion can also mean a change in ownership or an object, land, and so on.

In addition there is also the term unit conversion. conversion is changing the value of a system of units to the value of another unit. Unit conversion can be done in the same unit system or in a different unit system.

Unit conversion in the same system, for example, we want to change one unit in the international system to another unit in the same system. In this conversion unit conversion can use a conversion factor or with the help of a conversion ladder.
